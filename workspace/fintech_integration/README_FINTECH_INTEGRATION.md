# Stripe Billing & Multi-Currency Checkout Integration

---

## English

### Purpose
Integrates Stripe for global payments (EUR/USD/CNY), supporting Credit Cards, Alipay, and WeChat Pay. Includes subscription management, webhook for Supabase tier updates, localization, and premium API guard.

### Key Features
- Stripe Checkout: Multi-currency, multi-method, localized.
- Customer Portal: Manage subscriptions (Upgrade/Downgrade/Cancel).
- Webhook: Updates user's tier_id in Supabase after payment.
- Middleware: Authorization guard for premium endpoints.
