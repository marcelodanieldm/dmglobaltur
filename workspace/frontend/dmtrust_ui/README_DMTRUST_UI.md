# DM Trust Wallet & Transparency Ledger UI

---

## English

### Purpose
The DM Trust Wallet and Transparency Ledger UI are designed to make blockchain transactions clear and trustworthy for non-technical business owners.

### Key Features
- **Transaction Timeline:** Visualizes each step (Referral → Sale → Verification → Payout) in a clear, verifiable way.
- **Wallet View:** Shows balances in local currency (USD/EUR/CNY) alongside digital assets.
- **Alerts:** "Payout Successful" notifications with a clean, celebratory (Gold Matte) accent.
- **CEO View:** Liquidity Pool monitor displays circulating commission capital on the platform.

## Español

### Propósito
La interfaz de DM Trust Wallet y Transparency Ledger está diseñada para que las transacciones blockchain sean comprensibles y confiables para empresarios no técnicos.

### Características Clave
- **Línea de Tiempo de Transacciones:** Visualiza cada paso (Referencia → Venta → Verificación → Pago) de forma clara y verificable.
- **Vista de Billetera:** Muestra saldos en moneda local (USD/EUR/CNY) junto a los activos digitales.
- **Alertas:** Notificaciones de "Pago Exitoso" con un estilo limpio y acento dorado mate.
- **Vista CEO:** Monitor de Liquidez muestra el capital de comisiones circulando en la plataforma.
