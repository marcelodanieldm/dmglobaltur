# CEO Dashboard - Financial Settlements

---

## English

### Purpose
The CEO Dashboard provides real-time metrics on DM Global Tur's financial settlements, take-rate, and partner reliability.

### Key Features
- **Ecosystem Take-Rate:** Shows the percentage of revenue retained from transaction fees (0.5% - 1% per payout).
- **Global Settlement Clock:** Displays the average time from sale to payout (goal: under 1 hour).
- **MRR Chart:** Includes a new line for "Transaction Fee Revenue" alongside core revenue.
- **Partner Health:** Dashboard with credit scores for stores, highlighting reliability.
- **Regulatory Compliance:** Automated generation of "Proof of Commission" documents for tax purposes in multiple jurisdictions.

## Español

### Propósito
El CEO Dashboard muestra métricas en tiempo real sobre las liquidaciones financieras, la tajada (take-rate) y la confiabilidad de los socios de DM Global Tur.

### Características Clave
- **Take-Rate del Ecosistema:** Muestra el porcentaje de ingresos retenidos por comisiones de transacción (0.5% - 1% por pago).
- **Reloj Global de Liquidaciones:** Indica el tiempo promedio desde la venta hasta el pago (meta: menos de 1 hora).
- **Gráfico MRR:** Incluye una nueva línea para "Ingresos por Comisiones de Transacción" junto al ingreso principal.
- **Salud de Socios:** Panel con puntajes de crédito para tiendas, resaltando su confiabilidad.
- **Cumplimiento Regulatorio:** Generación automática de "Comprobante de Comisión" para impuestos en distintas jurisdicciones.
