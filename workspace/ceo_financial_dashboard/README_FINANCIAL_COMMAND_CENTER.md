# Financial Command Center for DM Global Tur CEOs

---

## English

### Purpose
Provides real-time visibility into MRR, CAC, LTV, and Net Burn Rate for strategic decision-making.

### Key Features
- MRR Growth Chart (Stripe + Supabase)
- Consumption Cost vs. Revenue (Gemini vs. User payments)
- Net Burn Rate metric
- CAC and LTV metrics
- Manual Override button to grant premium access
