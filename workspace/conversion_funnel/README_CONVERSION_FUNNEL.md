# Freemium Scoring Model & Lead-to-Cash Funnel

---

## English

### Purpose
Predicts which free users are most likely to convert to Premium (Tier 2) by tracking usage analytics and intent signals.

### Key Features
- Tracks Time-to-Value (how fast a user finds value after signup).
- Flags High Intent users (e.g., 3 city queries in 24h).
- Generates personalized Opportunity Report (Gemini 1.5 Flash API placeholder).

## Español

### Propósito

### Características Clave

- Marca usuarios de Alta Intención (ej: 3 consultas de ciudad en 24h).
- Genera Opportunity Report personalizado (placeholder para API Gemini 1.5 Flash).
