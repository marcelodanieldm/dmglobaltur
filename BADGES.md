# CI & Coverage Badges / Insignias de CI y Cobertura / CI kaj Kovro-Insignoj

---

## ENGLISH

Add these markdown badges to your main README for CI and code coverage status:

```
![CI](https://github.com/<your-username>/<your-repo>/actions/workflows/ci.yml/badge.svg)
![Coverage](https://img.shields.io/badge/coverage-unknown-lightgrey)
```

Replace `<your-username>` and `<your-repo>` with your GitHub info. For real coverage, integrate a tool like Codecov or Coveralls.

---

## ESPAÑOL

Agrega estas insignias markdown a tu README principal para mostrar el estado de CI y cobertura de código:

```
![CI](https://github.com/<tu-usuario>/<tu-repo>/actions/workflows/ci.yml/badge.svg)
![Coverage](https://img.shields.io/badge/coverage-desconocido-lightgrey)
```

Reemplaza `<tu-usuario>` y `<tu-repo>` por tu información de GitHub. Para cobertura real, integra Codecov o Coveralls.

---

## ESPERANTO

Aldonu ĉi tiujn markdown-insignojn al via ĉefa README por montri la staton de CI kaj kodo-kovro:

```
![CI](https://github.com/<via-uzantnomo>/<via-deponejo>/actions/workflows/ci.yml/badge.svg)
![Coverage](https://img.shields.io/badge/coverage-nekonata-lightgrey)
```

Anstataŭigu `<via-uzantnomo>` kaj `<via-deponejo>` per viaj GitHub-detaloj. Por vera kovro, integri Codecov aŭ Coveralls.
